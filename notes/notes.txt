How to push up information / documents to GitHub.
1.	cd to the project root (the name of the project directory)
2.	git status
3.	git add .
4.	git commit -m "message" (message is like an update... "updated notes / corrections made")
5.	git push origin main

(GIT Command) = 
git stash = removes or clears any changes to documents before you 'push' up to GitHub.
